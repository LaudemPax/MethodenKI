\section{Das Java-Projekt}

\subsection{IntelliJ IDEA}
Für die grafische Darstellung und die Verwaltung der Programmzustände wurde das Java Spieleentwicklungs-Framework LibGDX\cite{LibGdx} verwendet. Das Projekt wurde mit IntelliJ idea entwickelt und nutzt in hohem Maße das gradle build system. Das IntelliJ projekt kann wie folgt importiert werden:

\lstinline{File -> import -> $project-root-folder -> build.gradle}

Das Projekt wurde mit \textbf{openjdk-1.8} entwickelt und lässt sich am besten mit dieser Version von Java (Java 8) ausführen. Um das Projekt von Intellij Idea aus auszuführen, klicken Sie mit der rechten Maustaste auf die Klasse desktop/src/de.augsburg.hs.methoden.ki.DesktopLauncher und wählen Sie Run DektopLauncher.main. Die Verwendung von Intellij Idea ist nicht unbedingt erforderlich, jede Java-IDE, die \textbf{gradle unterstützt}, sollte ebenfalls funktionieren.

\subsection{Projekt Binary}

Das Projekt wurde auch in eine Jar-Datei exportiert. Die Datei ``methoden-KI.jar'' enthält das aus dem IntelliJ IDEA Projekt kompilierte Programm und kann verwendet werden, falls das Projekt nicht ausgeführt werden kann.

\begin{lstlisting}
java -jar methoden-KI.jar
\end{lstlisting}
